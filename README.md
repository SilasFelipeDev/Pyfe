# Pyfe
Criando o Jogo de baralho "Pife" em Python 

O Jogo Pife é um jogo de baralho, onde cada jogador recebe 9 cartas e tem como objetivo formarem 3 trincas.
As trincas podem ser 3 números iguais (7, 7, 7 por exemplo) de **naipes diferentes** OU uma sequência de 3 números diferentes (2, 3, 4 por exemplo) **com naipes iguais.**

O Jogo é por turno, ou seja, o jogador compra uma carta e descarta outra, em seguida outro jogador faz o mesmo. No ínicio, uma carta é puxada do baralho e exposta na mesa, as cartas do baralho ficam viradas com seu valor para baixo, as cartas na mesa ficam viradas com seu valor para cima, toda carta descartada vai diretamente para a mesa. O jogador pode escolher entre comprar do baralho ou comprar da mesa

Quem fizer 3 trincas primeiro, ganha.
